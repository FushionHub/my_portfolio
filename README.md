# Jamachi Mauricennadi – Animated Portfolio (Next.js)

This is an [Next.js](https://nextjs.org/) animated portfolio website created by **Jamachi Mauricennadi**, a web, mobile, and software developer. It was bootstrapped using [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

---

## 🚀 Getting Started

To get started with the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Then open your browser and visit:  
👉 [http://localhost:3000](http://localhost:3000)

---

## 🛠️ File to Edit

Start customizing your site by modifying:

```
app/page.js
```

Changes are applied in real-time thanks to hot-reloading.

---

## ✨ Features

- Built with Next.js and React
- Animated transitions using modern animation libraries
- Responsive design for mobile and desktop
- Font optimization via [`next/font`](https://nextjs.org/docs/basic-features/font-optimization) using Google Fonts (Inter)
