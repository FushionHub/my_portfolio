# Augustine Obi – Portfolio (Next.js)

This is a modern, animated portfolio website representing a core leader at **FushionHubAI.com.ng**, showcasing his unique expertise, services, and projects at the intersection of **marketing**, **operations**, and **software development**.

- **Augustine Obi** — Chief Marketing & Operations Officer (CMO/COO)  

---

## 💡 About the Project

Inspired by the sleek design of `me.fushionhubai.com.ng`, this site blends **business strategy** and **technical brilliance** into one seamless Next.js experience — highlighting results-driven marketing, intelligent operations, and powerful digital products.

---

## ⚙️ Technologies Used

| Technology     | Description                                               |
|----------------|-----------------------------------------------------------|
| Next.js        | React framework used for server-side rendering & routing   |
| Tailwind CSS   | Utility-first CSS framework for rapid UI development       |
| TypeScript     | Strongly-typed JavaScript for safer, scalable code         |
| Framer Motion  | Smooth animation library for engaging transitions          |
