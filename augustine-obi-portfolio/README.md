# Augustine Obi - Portfolio

This is the portfolio website for Augustine Obi, the CMO and COO of FushionHubai.com.ng. The website showcases his skills, services, and portfolio of work in the fields of marketing and operations.

The design of this website is inspired by `me.fushionhubai.com.ng`.

## Technologies Used

*   [Next.js](https://nextjs.org/) - A React framework for building server-side rendered and static websites.
*   [Tailwind CSS](https://tailwindcss.com/) - A utility-first CSS framework for rapid UI development.
*   [TypeScript](https://www.typescriptlang.org/) - A typed superset of JavaScript that compiles to plain JavaScript.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You will need to have [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/) installed on your machine.

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/your_username_/your_project_name.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```

### Running the Project

To run the project in development mode, use the following command:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Known Issues

During the development of this project, I encountered some issues with the development environment provided. Specifically, the `npm run dev` command would fail with a `next: not found` error, even though the `next` package was installed. This was due to the `node_modules/.bin` directory not being created correctly.

If you encounter this issue, you may need to run the development server with the full path to the `next` executable:

```bash
./node_modules/.bin/next dev
```
